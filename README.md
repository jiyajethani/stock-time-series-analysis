# Time Series Analysis with Stock Price Data

This project performs time series analysis on stock price data using Python.

## Features
- Trend visualization of stock prices
- Moving average smoothing
- Stationarity testing using Augmented Dickey-Fuller (ADF) test
- Differencing transformation for non-stationary data
- Autocorrelation and Partial Autocorrelation analysis

## Tools & Technologies
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Statsmodels  
yfinance

## Dataset
Stock price data downloaded using yfinance API.

## Project Workflow
1. Download stock price data
2. Visualize trend
3. Apply moving averages
4. Perform ADF test
5. Transform data using differencing
6. Analyze autocorrelation

Additional Feature
Stock price forecasting using ARIMA model to predict future trends.
